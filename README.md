# Event_Utsav_Symbiosis_Hackathon_2k26
Utsav is a comprehensive event management platform designed to streamline the planning, scheduling, and execution of institutional and cultural events. Built with a focus on efficiency and user experience, the project aims to replace manual coordination with an automated, data-driven approach.

Key Features
Intelligent Scheduling: Automated conflict detection to ensure no two events overlap in the same venue.

User-Centric Dashboard: Separate interfaces for administrators (organizers) and participants.

Real-time Notifications: Instant updates regarding venue changes, timing adjustments, or event cancellations.

Tech Stack
Frontend: HTML5, CSS3, JavaScript (React.js/Bootstrap)

Backend: Python (Flask/Django) or Node.js

Database: MySQL / PostgreSQL (for structured event and user data)

Tools: Git/GitHub for version control


Installation & Setup
To run this project locally, follow these steps:

Clone the repository:

Bash
git clone https://github.com/yourusername/utsav-project.git
Install dependencies:

Bash
pip install -r requirements.txt
Run the application:

Bash
python app.py
